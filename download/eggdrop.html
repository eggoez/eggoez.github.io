Sebelumnya saya memang tidak suka dengan bot, tapi untuk membantu segala sesuatu memang enak kalau ada bot, mencoba membuat bot yang satu paket sekali installasi langsung jadi, terlahirlah sebuah bot eggdrop? Apa nama bot itu? eggoez.tar.gz eggoez yang bisa dikatakan adalah eggdrop.goez. ketika sesekali anda main di iRC AllNetwork coba lihat eggoez!running@eggdrop.goez
Apa saja kelebihan eggoez bot ini?
sesuai yang saya katakan, saya hanya membuat bot yang instan 1 kali saji, ketika bot terinstall tidak perlu menambahkan sebuah TCL. Dan saya kira eggoez.tar.gz ini sudah memiliki kemampuan seperti ini:
dns | .dns6 | .dwhois | .ip | .port | .dnsnick | .host | .idle <nick> | .aksi | .saran | .hitung | .shio | .ping me | .jam | .google | .jodoh | .whois | .country | .cmd | mungkin untuk lebih lengkapnya ketika bot join silahkan ketik .help di channel :D
Bagiamana proses installasinya?
Proses installasi yang instan juga, saya tekadang heran, tapi tidak untuk sekarang, mungkin saya akan membuat bot online dengan htbash yang lebih cepat dan tidak terlihat ketika dilihat untuk ps-x atau prosesnya, ketika anda mendownload file installasi resmi eggdrop mungkin anda akan melakukan perintah seperti ini pada akhir, ./eggdrop -m nama.conf dan saya sudah mengakalinya dengan sesekali anda tidak repot untuk membuat file configurasi ataupun membuat daftar berupa setting yang ingin anda buat, kurang lebih seperti ini:

bagaimana cara install eggoez.tar.gz?

  1. Shell? Pasti.. Login terlebih dahulu
  2. Tambahkan file installasi pada directory anda, caranya adalah “wget http://eggoez.com/eggoez.tar.gz”
  3. Lalu anda extract file tersebut dengan “tar -zxvf eggoez.tar.gz”
  4. Setelah semua pindah ke directory eggoez “cd eggoez”
  5. Proses konfigurasi langsung ketik “./eggo 1 2 3 4 5 6”
    keterangan:
    1 = file konfigurasi yang ingin dibuat.
    2 = nick bot
    3 = ident bot
    4 = IP server Shell atau VPS anda
    5 = channel name ( jangan memakai # )
    6 = nick owner bot
    itu adalah keterangan dari perintah “./eggo” ex: ./eggo bot1.conf Bot1 Ones 178.33.140.138 ciut g0eZ
  6. Anda telah membuat konfigurasi “bot1.conf” jalankan dengan perintah ./goez <fileconf> ex: ./goez bot1.conf
  7. Tunggu beberapa saat bot akan muncul :D mudah bukan?

bagaimana cara menambah TCL?

menambah atau load tcl bila anda menggunakan eggdrop ini cukup mudah, anda bisa DCC bot setelah anda login ke bot langsung ketik `loadtcl nama.tcl ini untuk meload tcl dan pastikan nama.tcl yang ingin di load sudah ada para directory eggoez/scripts/ lalu untuk melihat tcl yang di load anda bisa ketik `listtcl atau apa ya, lupa saya, lebih jelasnya silahkan pv / msg bot dan ketik help nanti bot menerangkan secara details, lalu bila lewat SSH bagaimana? mudah, masuklah cd eggoez dan ketik ./tcl -t nama.conf nama.tcl ini tcl juga harus sudah ada pada dir eggoez/scripts/
./tcl : ini untuk perintah load tcl
-t : akan menambahkan sebuah scripts pada file nama.conf
nama.conf : adalah nama konfigurasi bot yang ingin ditambah tcl
nama.tcl : ini adala tcl yang ingin di load
sesudah sukses, lakukan rehash pada bot, bisa ketik -rehash pada channel atau pv bot dan command “rehash” jika tidak ada masalah bot akan rehashing, begitu juga sebaliknya, bila tcl / config ada yang no suck atau error bot akan quit atau keluar, untuk mengatasi masalah ini anda perlu kill proses bila masih berjalan dan run / menjalankan konfig bot tadi

bagaimana untuk kill atau die bot?

  1. Lihat bg prosesnya dengan melihat PID nya
  2. ketik ps -x bila sudah terlihat perintah yang dijalankan di kolom comand yg ada tulisan ex: ./goez bot1.conf langsung kill – 9 noPID
  3. Bila tidak ketik cat nama.pid untuk melihat PID nya ex : cat bot1.pid
  4. Setelah anda melihat server memberikan no pid nya kill dengan kill -9 ex: kill -9 123
